Git Commands Quickstart guide

Preface

Git is a source controller system which forms a tracking mechanism around a project entity (eg. a project folder).

Git init

Defines a directory for git so that it sets up its configuration inside a hidden folder.

Git status

Returns information about current commits as well as a list of untracked files within the work directory and its sub-directories.

Git add 'parameter'

Adds files to the tracker within the project entity. Git add . adds all files.

Git commit 'parameter'

Submits current files to HEAD with a name tag Git commit -m "name" commits files with the name tag.

Git branch

Lists the current branch.

Git log

Lists commits.
